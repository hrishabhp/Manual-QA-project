## Bug ID: ORG_BUG_01

**Title:** Error message is not clearly highlighted on invalid login  

**Module:** Login  

**Severity:** Medium  
**Priority:** Medium  

---

### Description
When a user enters invalid login credentials, the system displays an error message, but it is not clearly highlighted, which may confuse first-time users.

---

### Steps to Reproduce
1. Open the OrangeHRM demo application  
2. Enter an invalid username and password  
3. Click on the Login button  

---

### Actual Result
An error message is displayed but it is not clearly visible or highlighted.

---

### Expected Result
The error message should be clearly visible and highlighted so users can easily understand the issue.

---

### Environment
- Browser: Google Chrome  
- Operating System: Windows 10
